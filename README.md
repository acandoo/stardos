# stardos

`stardos` is a batteries-included, cross-runtime library for accessing operating system resources in Gleam!

[![Package Version](https://img.shields.io/hexpm/v/stardos)](https://hex.pm/packages/stardos)
[![Hex Docs](https://img.shields.io/badge/hex-docs-ffaff3)](https://hexdocs.pm/stardos/)

# Installation

You can add `stardos` to your command line as follows:

```sh
gleam add stardos
```

```gleam
import stardos/concurrent/future.{type Future}
import stardos/concurrent/task
import stardos/env

pub fn main() -> Nil {
  // TODO
}
```

Further documentation can be found at <https://hexdocs.pm/stardos>.
